Multi-Genre Engine Platform

Any genre. Any dimension. One studio.

Ketebe ships with five production-ready engines today — and is expanding to cover every major game genre. Pick your engine, and the tooling, physics, and editors follow automatically.

◈

IsoPixel 2.5D

Isometric pixel art engine with depth-sorted rendering, a dynamic day/night cycle, particle systems (IsoFX), and procedural vegetation. Built for tactical RPGs and simulation games.

  • Depth-sorted tile rendering
  • Day/Night atmosphere system
  • IsoFX particle system
  • NPC behavior scripting
⬡

RPG Top-Down 2D

Systems-heavy engine with branching dialogue trees, quest management, inventory, NPC schedules, and a dynamic logic scripting runtime. Great for story-driven RPGs.

  • Branching dialogue trees
  • Quest & achievement system
  • Inventory management
  • JSON/JS logic scripts
▶

Platformer 2D 2D

Precision physics engine with AABB collision detection, smooth camera interpolation, and a frame-perfect character animator. Tight, responsive gameplay is the priority.

  • AABB collision system
  • Smooth camera interpolation
  • Frame animator
  • Visual platformer editor
⬛

TopDown 3D 3D

Top-down perspective with real 3D depth. Includes an ability system, fog of war, minimap, and full A* pathfinding. Bridges pixel aesthetics with 3D immersion.

  • 3D depth rendering
  • Ability & entity systems
  • Fog of war + minimap
  • A* pathfinding
◎

FPS 3D 3D

First-person engine with custom 3D camera, enemy AI, decal systems, and adapter-based rendering. For retro-aesthetic first-person experiences.

  • First-person 3D camera
  • Enemy AI system
  • Decal system
  • FPS3D adapter

Kai — Built-in AI

Your AI pair programmer, without the cloud

Kai is Ketebe's local AI assistant. It runs entirely in your browser using WebGPU and WASM — no API keys, no internet connection required, no data leaving your machine.

⚡
Local inference

Powered by Transformers.js running on WebGPU/WASM. Everything executes on your hardware.

🔍
Project-aware RAG

Kai indexes your project files, logic scripts, and engine documentation using a local vector store (Orama) for context-aware answers.

🛠
Agentic capabilities

Kai can create files, modify logic scripts, and perform tasks inside your project — not just suggest them.

🔒
100% private

No cloud, no telemetry, no API subscriptions. Your project code never leaves your machine.

Multi-Platform

Build once, ship everywhere

One command packages your game for any target. The same project runs on desktop, web, and mobile without changes.

🖥

Desktop

Ships as a standalone Windows app via Electron. Embeds the game server internally — no separate install needed.

npm run build:game "MyGame" electron
🌐

Web

Serves directly from an Express static server. The same HTML/JS game runs in any modern browser.

npm run build:game "MyGame" web
📱

Android

Packaged via Capacitor. Sync assets and open Android Studio with a single command.

npm run build:game "MyGame" android
🍎

iOS

Also via Capacitor. Builds the iOS project and opens Xcode ready for deployment.

npx cap open ios

Primal Simplicity

Primitive instinct. Modern power.

Even a caveman can build a world with Ketebe. We've distilled the complexity of game development into an experience so natural, it feels like muscle memory. Get up and running in three steps.

1

Clone and install

git clone https://github.com/n0rthstar/ketebe.git
cd ketebe
npm install
2

Launch the studio

# Desktop app (Electron)
npm start

# Or in your browser
npm run server
# → open http://localhost:3000
3

Create or open a project

From the project dashboard, pick one of the included demo projects — Default Project (RPG), Platformer Demo, or Iso Pixel Demo — or create a new project and choose your engine type.
