  DATA FLOW & RECENT EVOLUTIONS:
  1. DUAL-VIEWPORT SYNC: Editing in the 2D Floor Plan (Canvas) generates voxel diffs pushed to the 3D THREE.js Scene.
  2. PROJECT OVERLAY: Server serves files from `projects/ActiveProject/` if they exist, otherwise `public/` (Core).
  3. AI AGENTIC LOOP: AI Chat UI (Phase 2 MSN-Retro) -> Orama RAG -> Tool Registry -> EventBus -> IDE Command.
  4. HYBRID RAG: Local Orama Vector Store (IndexedDB) for docs; Python backend handles large model inference/heavy vector pkls.
  5. RENDERER RESIZE FIX: Renderer3D now implements a public `.resize(w, h)` method used by all 3D engine adapters.
  6. ASSET PATH FIX: Code Forge and AssetManager now implement client-side path sanitation to prevent double-prefix 404s.
  7. ISOPIXEL V8: Removed occlusion culling for better sprite stability; added dynamic day/night light progression.
