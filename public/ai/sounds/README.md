# MSN/AOL Messenger Sounds

This directory contains authentic messenger sounds for the IRAB MSN interface.

## Sound Files Needed

Place these sound files here (MP3 or WAV format):

### MSN Messenger Sounds:
- `message-received.mp3` - Classic MSN "door opening" sound when message arrives
- `message-sent.mp3` - Soft "whoosh" when sending message
- `online.mp3` - Sound when IRAB comes online
- `nudge.mp3` - MSN nudge/wink sound
- `typing.mp3` - Typing indicator sound (optional)
- `error.mp3` - Error notification sound

### AOL Instant Messenger Sounds:
- `aol-welcome.mp3` - "Welcome!" voice
- `aol-goodbye.mp3` - "Goodbye!" voice
- `aol-message.mp3` - Classic AIM door slam
- `aol-buddy-on.mp3` - "Buddy online" sound
- `aol-buddy-off.mp3` - "Buddy offline" sound

## Sound Sources

You can find authentic messenger sounds at:
- Archive.org MSN Messenger sound packs
- WinCustomize.org classic sounds
- YouTube "MSN Messenger sounds" (use youtube-dl to extract audio)
- https://www.sounds-resource.com/

## Usage

The sounds are automatically played by `assistant-panel.js` when:
- Messages are sent/received
- Chat window opens
- Errors occur
- IRAB starts typing (thinking)

Configure sounds in Settings panel (⚙) → Enable/Disable sounds
