## Configuration

For Android Cerebras API, create `.redglitch/ai_config.json`:

```json
{
  "provider": "cerebras",
  "model": "llama3.1-8b",
  "cerebrasKey": "your-api-key-here"
}
```

Desktop/web uses local inference automatically.
