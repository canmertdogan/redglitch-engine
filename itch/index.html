About

RedGlitch is a professional-grade, open-source game development studio. Unlike generic game engines, RedGlitch ships multiple specialized engines — each tuned for a specific genre — so you're not fighting a one-size-fits-all tool. Whether you're building an isometric tactical RPG, a precision platformer, a story-driven top-down adventure, or a retro FPS, there's an engine ready to go.

Every engine comes with its own set of integrated visual editors — over 30 in total. Edit dialogue trees, design levels, paint sprites, script behaviors, compose music, manage quests, and more — all from within the studio. No context switching, no external tools required.

On top of that, Kai AI — a fully local, privacy-first AI assistant — runs directly in your browser via WebGPU/WASM. Kai can answer questions about your project, generate code, modify files, and act as your pair programmer, all without sending data to any server.

RedGlitch supports 4 target platforms: Desktop (Electron), Web (Express), Android (Capacitor), and iOS (Capacitor). Build once, ship everywhere with a single command.

Getting Started

1. Clone the repository: git clone https://github.com/n0rthstar/redglitch.git

2. Install dependencies: npm ci

3. Launch the studio: npm start

4. Or run the web server: npm run server

Open your browser to http://localhost:3000 and start building.
