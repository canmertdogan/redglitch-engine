# Pixel Art Editor
**Category:** ASSETS
**Tool ID:** `pixel`

### Purpose
A dedicated tool for creating and editing pixel art sprites directly within the Studio.

### Key Features
- **Grid Editing:** Precision pixel placement.
- **Palette Management:** Create and save color palettes.
- **Sprite Slicing:** Prepare sprite sheets for engine use.
- **Immediate Preview:** See how sprites look in the engine's pixel-perfect renderer.

### AI Usage
The AI can open this tool using `[[openTool:pixel]]`. Use this for art or sprite modification requests.
