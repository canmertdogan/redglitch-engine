# Quest Designer
**Category:** LOGIC & AI
**Tool ID:** `quests`

### Purpose
The Quest Designer is used to create structured story objectives, rewards, and branching narratives.

### Key Features
- **Objective Tracking:** Define quest steps and completion criteria.
- **Reward System:** Assign items, XP, or flags upon completion.
- **Visual Flow:** View quest chains and dependencies.

### AI Usage
The AI can open this tool using `[[openTool:quests]]`. Use this for narrative and progression requests.
