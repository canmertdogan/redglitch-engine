# IsoPixel Studio
**Category:** WORLD ARCHITECT
**Tool ID:** `iso_studio`

### Purpose
IsoPixel Studio is a specialized isometric 2.5D editor for creating detailed pixel art worlds with depth.

### Key Features
- **Isometric Grid:** True isometric perspective rendering.
- **Elevation Control:** Place blocks and props at different heights.
- **Vegetation Generation:** Integrated system for procedural grass, trees, and bushes.
- **NPC Placement:** Specific support for isometric character sprites and paths.
- **Day/Night Cycle:** Dynamic lighting preview for isometric environments.

### AI Usage
The AI can open this tool using `[[openTool:iso_studio]]`. It is ideal for "isometric" or "depth-based" level requests.
