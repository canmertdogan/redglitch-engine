# Level Editor
**Category:** WORLD ARCHITECT
**Tool ID:** `editor`

### Purpose
The Level Editor is the primary tool for designing 2D top-down and standard RPG environments in the Vortex Engine.

### Key Features
- **Tile Placement:** Grid-based placement of environment tiles.
- **Collision Mapping:** Define walkable and blocked areas.
- **Entity Spawning:** Place NPCs, items, and event triggers directly into the world.
- **Multi-layer Support:** Separate layers for ground, decorations, and foreground elements.

### AI Usage
The AI can open this tool using `[[openTool:editor]]`. Use this for general level design requests.
