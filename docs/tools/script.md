# Script Editor
**Category:** LOGIC & AI
**Tool ID:** `script`

### Purpose
The Script Editor provides a full-featured code environment for writing game logic using JavaScript.

### Key Features
- **Monaco Editor Integration:** Modern IDE features like syntax highlighting and autocomplete.
- **Engine API Access:** Direct access to RedGlitch Engine core classes (Player, Entity, Physics).
- **Auto-save:** Logic is persisted as you write.
- **Project Scope:** Automatically focuses on the scripts within the current active project.

### AI Usage
The AI can open this tool using `[[openTool:script]]`. It is the primary tool for code-related tasks.
