# Asset Manager
**Category:** ASSETS
**Tool ID:** `assets`

### Purpose
The Asset Manager is the central file browser for the Studio, used to organize, preview, and manage project resources.

### Key Features
- **File Organization:** Folder-based management of sprites, sounds, and data.
- **Dependency Tracking:** Automatically maps how files relate to each other.
- **Metadata Viewer:** View file sizes, modification dates, and engine-specific tags.

### AI Usage
The AI can open this tool using `[[openTool:assets]]`. Use this for file management requests.
