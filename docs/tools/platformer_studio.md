# Platformer Studio
**Category:** WORLD ARCHITECT
**Tool ID:** `platformer_studio`

### Purpose
The Platformer Studio is a high-performance 2D side-scrolling level editor featuring advanced physics and procedural generation.

### Key Features
- **Advanced Physics:** Native support for 45° slopes, one-way platforms, and moving platforms.
- **Smart Generator:** Integrated `JumpSimulator` ensures all generated parkour is reachable by the player.
- **Theme Strategies:** Supports themes like Flow, Spire, Abyss, Gauntlet, and Clockwork.
- **Chunk Caching:** Optimized rendering for large, complex levels.

### AI Usage
The AI can open this tool using `[[openTool:platformer_studio]]` or generate content directly using `[[generateParkour:theme,difficulty]]`.
