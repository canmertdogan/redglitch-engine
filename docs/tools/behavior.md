# AI Behavior Editor
**Category:** LOGIC & AI
**Tool ID:** `behavior`

### Purpose
This tool allows developers to design complex logic flows and "Brains" for entities using a behavioral state machine or logic trees.

### Key Features
- **Visual Logic:** Design AI decision-making processes.
- **State Management:** Define Idle, Aggressive, Patrol, and other behaviors.
- **Sensor Setup:** Configure how AI detects the player or environment.

### AI Usage
The AI can open this tool using `[[openTool:behavior]]`. Use this for AI and logic complexity requests.
