# Audio Studio (DAW)
**Category:** ASSETS
**Tool ID:** `daw`

### Purpose
The Audio Studio is a workstation for managing sound effects, background music, and audio triggers.

### Key Features
- **Sound Management:** Upload and categorize audio files.
- **Loop Points:** Define where music segments repeat.
- **Audio Mixers:** Adjust gain and effects for engine playback.

### AI Usage
The AI can open this tool using `[[openTool:daw]]`. Use this for sound or music requests.
