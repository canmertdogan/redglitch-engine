# NPC Editor
**Category:** ENTITIES
**Tool ID:** `npc`

### Purpose
The NPC Editor allows for the creation and configuration of non-player characters, including their visuals and basic behaviors.

### Key Features
- **Visual Assignment:** Select sprites and animations for NPCs.
- **Dialogue Linking:** Connect NPCs to conversation trees.
- **Stats Management:** Set health, speed, and other RPG parameters.
- **Spawn Logic:** Define how and where NPCs appear in the world.

### AI Usage
The AI can open this tool using `[[openTool:npc]]`. Use this for character-related requests.
